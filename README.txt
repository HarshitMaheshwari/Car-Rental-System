Steps to run the Car Rental System:
1. Run the main.py file.
2. You will get a Welcome message and a the count for the total number of cars available in the system.
3. After the welcome message, there will be a prompt to ask the user if they want to rent a car or return a previously rented car.
4. If the user choose to rent a car, then a prompt will ask for the category of car that they are interested in.
5. After selecting an appropriate category, A list of all the available cars in that category will be displayed with all the details of the cars like price, carID, name. And a prompt will ask the ID of the car that you want to rent.
6. After entering the car ID, it will ask to enter the customer ID. There are two customer IDs available. Customer ID for General customer: 100;  Customer ID for Vip customer: 101.
7. A message will be displayed. After which the system will ask for the booking start date in a specified.
8. After entering the booking start date, it will ask for the booking end date.
9. A booking confirmation prompt will ask for the confirmation for the selected dates.
10. After confirming the dates, a final booking confirmation prompt will appear.
11. Confirming this prints the confirmation message.

To return a previously rented car:
12. It will ask for the customer ID. There are two customer IDs available. Customer ID for General customer: 100;  Customer ID for Vip customer: 101.
13. After confirming the customer ID, Bill gets displayed. 
14. A prompt asks for the final payment. And after payment, the system starts from the start. While displaying the total number of cars available.
